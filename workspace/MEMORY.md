## Effective Frameworks

### Business
- **JTBD (Jobs To Be Done)** — Ask "what job is the user hiring this for?" instead of "who is the user?"
- **Minimum Viable Test** — Find the cheapest way to validate each direction before building
- **Leverage Points** — What existing assets can be leveraged?

### Technical
- **YAGNI** — Cut features you "might need later"
- **Design First** — No code until design is approved
